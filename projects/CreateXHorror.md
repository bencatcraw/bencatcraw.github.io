## CreateX Lab
The CreateX Lab is a room in the ACM building at the University of West Oahu campus. There are an array of projectors that connect to a powerful computer and project unity scenes onto the walls and floor of the room to create immersive experiences.

## The Project
The second project from my AR/VR Development class was to create an interactive experience using the CreateX Lab. This was a stressful project as many times the packages in use to develop for the lab corrupted my unity project, but in the end we have a relatively engaging 1-2 minute long experience. The player uses a microsoft kinect tracker to track their body position which allows them to move a flashlight around the scene and experience spooky interactions.
