## This essay was written for my Software Engineering Class

So far, I'm finding Javascript to be a relatively easy language to understand. I have a lot of experience with other languages like C, C#, C++, and Java, so many aspects are familiar to me. One thing I dislike about Javascript is remembering the different variable types and what they are useful for. In other languages I usually declare variables using types like int, char, string, bool, etc, which tell you what type of data it will store. Javascript uses var, let, and const which can store any type of data. I think this provides a more dynamic way of creating variables though, which is better for software engineering. For me, the practice WODS have been super useful and they help me realize where my understanding is at with javascript. I think they are an incredibly useful way of learning and can even be fun to race against the clock.
