Expanding on Software Engineering

11 Dec 2023

Introduction

Software Engineering principles are not only important to software engineering. In my pursuit to become a game designer/programmer, I find I can apply many of these principles from software engineering to game design.

Agile Project Management

I learned a lot about project management from this class that would apply well to my future production of games. One thing I’ve struggled with in the past working on a game is keeping tasks and people organized in a productive way. Now after completing this class, I have new angles and direction for project management in the future whether it be on games or other software development projects.

Design Patters

Another thing I learned from this class was design patterns. The idea of design patterns covers a large blanket of functionality in a program, but some of the most important one I can think of that I’ve already used in other projects after learning about in this class are builder classes, state changes, observers, subscriptions, etc. All of these and more have been crucial to this class and my own personal projects and I hope to use them in the future as well.
