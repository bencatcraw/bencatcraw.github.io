## My thoughts on Bootstrap

So far, I have found Bootstrap 5 to be a really useful tool for creating websites. It was initially confusing navigating bootstrap because there is a lot of stuff it doesn't autocomplete and you have to search the documentation. The documentation for Bootstrap though is very well written and thourough and has helped me through many of the assignments in my software engineering class.

## Web Dev and UI Frameworks

UI frameworks are very powerful tools that offer many benefits over hard coding everything from scratch in html. They have a small investment period to learn, but once learned they improve development speed, consistency, functionality, and scalability. I have enjoyed using Bootstrap, but I am excited to try other UI frameworks in the future.
